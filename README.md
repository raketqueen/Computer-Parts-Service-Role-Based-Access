# PC Hardware Inventory

A simple inventory management system for PC parts built with Flask and MySQL. 

## Features
- Add, edit, delete parts
- Search and filter by category
- Light/Dark mode
- Responsive UI with Bootstrap
- Added the Role Based Access for users
- Clean the view in main page
- Hashed the password of the user in MySQL and init.sql
- Added editor and viewer role


## Setup
1. Install Python 3.x and MySQL
2. Create a database named `partsdb`
3. Install dependencies:

```bash
pip install -r requirements.txt
